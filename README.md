# Requirements
Nodejs version > 14.
Installed chrome

# Install packages
> yarn install
> 

Copy .env
> cp .env .env.example
> 
Change param APP_PORT at .env (Default is 9000)

# Run code
> yarn dev
> 

# Launch page
Go to localhost::9000 Or localhost::{PORT}
> PORT is the number you added .env 

# Run demo crawl data
Go to localhost::9000/crawl/crawl-list-page
Checking terminal
Checking file logs/all.log
